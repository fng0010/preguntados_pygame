## Pasos a seguir

#### Crear un ambiente virtual
```bash
# Linux
python3 -m venv venv

# MacOS, Windows
python -m venv venv
```

#### Activar ambiente virtual
```bash
# Linux, MacOS
source venv/bin/activate
```

```powershell
# Windows (Powershell)
.\venv\Scripts\Activate.ps1
```

#### Instalar dependencias
```bash
pip install -r requirements.txt
```

#### Correr el juego
```bash
python3 main.py
```
